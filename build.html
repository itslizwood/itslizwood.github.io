Twitter API

Working with my thesis partner, we created a sentiment analysis of Twitter using colour.


An Arduino was used to connect a string of LED lights to my laptop. We wrote the program in Processing in order to use Java and tap into Twitter's API that grabs a handful of tweets from the pipeline.


We mapped specific keywords to several different colours that would express sentiment towards the Toronto Raptors basketball team (they were in the playoffs at the time). When we ran the program, the LED's would light up based on the tweets coming in.
